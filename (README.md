# 本地音频剪辑器

一个完全在浏览器中运行的音频剪辑工具，无需上传文件到服务器，保护您的隐私安全。

## 功能特性

### 核心功能
✅ **完全离线运行** - 所有处理均在您的浏览器中完成  
✅ **可视化波形编辑** - 拖拽选区或精确输入时间  
✅ **批量处理** - 支持多个文件同时裁剪  
✅ **格式支持** - 支持 MP3, WAV, OGG, FLAC, AAC, M4A 等格式  
✅ **本地存储** - 处理结果直接保存到您的电脑  
✅ **打包下载** - 支持将多个文件打包为ZIP下载

### 高级功能
🎵 **音频效果**
- 淡入淡出效果 - 自定义淡入/淡出时长
- 音量标准化 - 自动调整音量到最佳水平
- 实时波形显示 - 可视化查看音频

⌨️ **快捷键支持**
- Space - 播放/暂停
- 左右箭头 - 调整选区起始位置
- Ctrl+S - 快速开始裁剪

🎯 **用户体验**
- 拖拽上传支持
- 实时进度显示
- 当前文件高亮
- 批量处理进度跟踪  

## 使用方法

1. **直接打开**：双击 `index.html` 文件在浏览器中打开
2. **上传文件**：点击上传区域或拖拽音频文件到页面
3. **选择片段**：在波形图上拖拽选择，或输入精确时间
4. **开始裁剪**：点击"开始裁剪"按钮
5. **下载结果**：单个文件自动下载，或使用"打包下载"

## 技术说明

- 使用 **Web Audio API** 进行音频处理
- 使用 **wavesurfer.js** 显示波形
- 使用 **JSZip** 打包下载
- 纯前端实现，无需后端服务器

## 浏览器兼容性

- Chrome 60+
- Firefox 60+
- Safari 14+
- Edge 79+

## 注意事项

- 大文件处理可能需要较长时间
- 输出格式目前主要为WAV，后续可集成更多编码器
- 建议使用现代浏览器以获得最佳体验

## 开发说明

如需二次开发，直接编辑以下文件：

- `style.css` - 样式设计
- `app.js` - 主应用逻辑
